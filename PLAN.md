# 걸어볼까 - 기획서 v2

## 1. 앱 개요

**앱 이름:** 걸어볼까
**한줄 설명:** 내 주변 산책로와 벚꽃/단풍을 지도에서 보고, 친구들과 산책을 공유하는 앱
**마스코트:** 배낭 멘 곰

**핵심 가치:**
- 사용자가 아무것도 안 해도 데이터가 알아서 가치를 줌
- 지도 중심으로 한 화면에서 모든 정보 확인
- 인스타 느낌의 사진 공유 + 장소 태그 커뮤니티

## 2. 타겟 사용자

- 산책/등산 즐기는 2030
- 벚꽃/단풍 시즌에 명소 찾는 사람
- 이사할 때 동네 환경 궁금한 사람 (소음지도 - 별도 앱)

## 3. 수익 모델

- 기본 무료 + 프리미엄 구독 (상세 분석, 알림)
- 지역 광고 (카페, 음식점)
- 부동산 중개업소 제휴

## 4. 앱 계획

### 앱 1: 걸어볼까 (현재 개발 중)
- 산책로/등산로 + 벚꽃/단풍 + 커뮤니티

### 앱 2: 소음지도 (별도)
- 동네별 소음 레벨 히트맵
- 이사/원룸 참고용

### 앱 3: 주차 난이도 맵 (이후)

### 앱 4: 축제/이벤트 레이더 (이후)

## 6. 기술 스택

## 7. 데이터소스

| 데이터 | API | 상태 |
|--------|-----|------|
| 날씨 (기온/습도/풍속/강수) | 기상청 단기예보 | 연동 완료 |
| 미세먼지/대기질 | 에어코리아 | 연동 완료 |
| 산책로/등산로 | 한국관광공사 KorService2 | 연동 완료 |
| 벚꽃/단풍 일기온 | 기상청 ASOS | 연동 완료 (키 승인 필요) |

| 커뮤니티 (사진/리뷰) | 자체 DB | 예정 |
